# 〈 ทำไม · Why 〉

<br>

> **เพราะของพื้นฐานในชีวิต**
> — เวลา ร่างกาย เงิน ข้อมูล —
> **คือของที่ต้องอยู่ในมือเราเอง**

<br>

> **Because the basics of a life**
> — time, body, money, data —
> **must remain in our own hands**
